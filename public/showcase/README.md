# Showcase images

Drop screenshots of real apps/dashboards/landing pages here using these filenames
(matching the slugs in `lib/content.ts → SHOWCASE`):

- `landing.png` — landing page screenshot
- `dashboard.png` — dashboard screenshot
- `miniapp.png` — mini-app screenshot
- `home.png` — home/family system
- `business.png` — small business system
- `automation.png` — automation flow

Recommended size: ~1200×900px (4:3 aspect). PNG or JPG both fine.

If a file is missing, the page renders a polished SVG mockup (browser/dashboard/phone)
in its place so the layout never breaks.
